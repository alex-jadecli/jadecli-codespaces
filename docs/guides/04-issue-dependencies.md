# Issue Dependencies

> Define blocking relationships between issues to track bottlenecks.

## Key Concepts
- Two relationship types: "blocked by" and "blocking"
- Blocked issues show icon on project boards and Issues page
- Manage via Relationships sidebar on issue page
- Requires triage permissions
- Available on Free, Pro, Team, Enterprise Cloud

## CLI Commands
```bash
# No CLI support for dependencies
# Manage via web UI: Issue > Relationships sidebar
```

## Limits
- No documented limits on dependency count
