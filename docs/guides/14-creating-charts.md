# Creating Charts

> Visualize project data with insights and charts.

## Key Concepts

- Charts visualize project progress and metrics
- Filter charts to show specific data subsets
- Share insights with stakeholders
- Access via insights icon (graph) in project

## Creating a Chart

1. Navigate to project
2. Click insights icon (top-right graph)
3. Click "New chart" in left menu
4. Optionally rename via dropdown
5. Add filters above chart
6. Click "Save changes"

## Chart Configuration

- Add filters to customize data displayed
- Rename charts for clarity
- Multiple charts per project
- Filter syntax matches project filters

## Chart Types

- Current data charts
- Historical/burn-down charts (if available)
- Status distribution
- Assignee workload

## Filtering Charts

- Use same filter syntax as views
- Example: `status:done iteration:@current`
- Filters reduce dataset shown in chart
